# Contributing

This is a solo-operated project. PRs are welcome but may be closed without
merging. If you want to suggest a change, open an issue first so we can
discuss whether it fits the project direction.

## Before opening a PR

- Bug fixes and typo corrections in content are the most likely to be accepted.
- New features or design changes will almost certainly require a discussion first.
- Please do not open PRs that change brand assets, fonts, spacing, or colors.

## Local development

```bash
cp .env.example .env.local   # fill in your own Firebase project values
npm install
npm run dev
```

Requires Node 22. See README.md for full setup instructions.
