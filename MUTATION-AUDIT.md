# Mutation Audit

Not yet run. This file is regenerated automatically by the weekly
`.github/workflows/mutation-audit.yml` scheduled workflow (Mondays 6am UTC),
which opens a draft PR with the updated summary and an attached HTML report
artifact. It is informational only and never auto-merges: mutation testing
finds weak or missing assertions, but a human still has to judge each
finding, since some surviving mutants are equivalent mutants with no real
behavior difference rather than real gaps, before writing a test.

Trigger a run early via GitHub → Actions → Weekly Mutation Audit → Run workflow.
