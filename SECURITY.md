# Security Policy

## Reporting a vulnerability

Please do not open a public GitHub issue for security vulnerabilities.

Email **security@garammasaladating.com** with:

- A description of the vulnerability and its potential impact
- Steps to reproduce or a proof-of-concept
- Any suggested remediation if you have one

You can expect an acknowledgment within 48 hours and a resolution timeline
within 7 days for critical issues.

## Scope

- garammasaladating.com and all subdomains
- The Firestore database and Firebase Storage bucket
- Admin and contestant-prep authenticated routes
- API endpoints under /api/

## Out of scope

- Issues in third-party services (Firebase, Vercel, Eventbrite, Resend)
- Theoretical vulnerabilities without a practical exploit path
- Denial-of-service attacks
- Social engineering

## Disclosure policy

We follow coordinated disclosure. Please allow us time to patch before
publishing details publicly.
